Ayuda

Bienvenido al panel de configuración del Sistema Experto.

¿Qué es un Sistema Experto? Un programa que usa reglas del tipo "SI condición ENTONCES acción" para tomar decisiones, imitando el razonamiento de un experto humano.
¿Qué es Lógica Difusa? En lugar de valores "sí/no" estrictos, usa grados de pertenencia (0 a 1). Por ejemplo, 37°C puede ser "normal" con 0.7 y "cálido" con 0.3 al mismo tiempo.
Reglas Ej: SI vibración=alta ENTONCES prioridad=crítica. Las condiciones usan sensores y términos difusos. El consecuente es el nivel de alerta.
Sensores Cada sensor tiene un rango (ej: 0-10) y términos difusos con funciones de membresía (trimf, trapmf). Los parámetros definen la forma de la curva.
Funciones de Membresía trimf(a,b,c) = triangular. trapmf(a,b,c,d) = trapezoidal. Parámetros definen puntos en el rango del sensor. Ej: trimf(0,0,4) = "bajo" de 0 a 4.
Colores de estado Verde = Normal/Bajo · Amarillo = Medio · Rojo = Alto · Morado = Crítico

Cada condición es del formato: [nombre_del_sensor] es [término_difuso]. Ej: vibration es high. Use los sensores disponibles del activo (ver pestaña Sensores).

Nuevo Sensor

Un sensor mide una variable física del activo. Las funciones de membresía definen cómo se clasifican sus valores en términos difusos (bajo, medio, alto, etc.).
Cada término es una categoría (ej: "bajo", "medio", "alto"). Los parámetros definen la forma de la curva:
trimf(a,b,c) = triangular: en a empieza, en b está al 100%, en c termina.
trapmf(a,b,c,d) = trapezoidal: plano entre b y c.
Ejemplo para sensor de 0 a 10: bajo: trimf(0,0,4), medio: trimf(2,5,8), alto: trimf(6,10,10)
